# MERN - CRUD

MongoExpressReactNode - CreateReadUpdateDelete boilerplate

# How to start

 1. Start **mongodb** service
 2. in *root* folder run '**yarn install**' and run '**yarn start**'
 4. in */api/* folder run '**yarn install**' and run '**nodemon server**'
